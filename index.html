How to Use

  1. Browse Applications: View available PowerShell installation scripts in the grid below.
  2. Check Status: Green checkmark (‚úì) means installed, red X means not installed.
  3. Select Apps: Check the boxes for applications you want to install.
  4. Generate Command: Click "Generate Command" to create the PowerShell command.
  5. Run Command: Copy and run the generated command in PowerShell as Administrator.

Features

  • Real-time Validation: Automatically checks if applications are already installed
  • Search: Use the search box to quickly find specific applications
  • Accessibility: Full keyboard navigation and screen reader support
  • Responsive Design: Works on desktop, tablet, and mobile devices

Requirements

  • Windows PowerShell 5.1 or PowerShell 7+
  • Administrator privileges for installations
  • Internet connection for downloading applications
